Anaconda

Anaconda 설치시 자동으로 Python도 설치됨Anaconda : python 프로그래밍에 유용한 툴 패키지

Anaconda Navigator 를 통해 Jupyter 실행시 python 코드를 작성할 수 있는 노트북 페이지가 나온다

Jupyter 사용법

  • 초록색 테두리 : 입력모드
  • 파란색 테두리 : 명령모드
  • esc 키로 [ 명령모드 ] 진입
  • 명령모드일 때, 'h' 누르면 단축키 확인

Jupyter 단축키

  • a : 위에 셀 추가
  • b : 아래에 셀 추가
  • d (두 번) : 셀 삭제
  • ctrl + enter : 해당 셀 실행만 하고 아무것도 안함
  • shift + enter : 해당 셀 실행 후 아래 셀로 이동 (아래에 셀이 없다면 셀 생성 후 이동)
  • shift + tab : 함수에 대한 설명 표시
  • m : markdown 타입으로 전환
  • y : code 타입으로 전환

셀의 타입

셀은 타입이 존재한다

  • Input 타입 (default) : python 코드를 입력할 수 있는 타입. [] 안의 숫자는 몇 번째 실행인지를 표시해줌
  • Markdown 타입 : 문서의 내용과 구조를 한꺼번에 명시할 수 있는 것. 코드에 한 설명을 적어줄 수도 있다
  • # TITLE1 : 가장 큰 타이틀 (##, ###, ... # 이 적을수록 큰 타이틀)
  • **특정단어** : 특정단어만 굵게
  • *특정단어* : 특정단어만 기울게
  • - : 비순서형 리스트(들여쓰기 한 칸 할 때마다 depth 추가)
  • #. : 순서형 리스트
  • > 인용구 : 인용구
  • $수학식$ : 수학식 폰트 적용
  • ```language : 코드로 인식
    ```python
    코드
  • [링크이름] (url) : 링크 이름 바꾸기
