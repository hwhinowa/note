Props와 State

React Component에서 다루는 데이터는 props와 state 두 가지

Props

부모 Component가 자식 Component에서 주는 값이 때 자식 Component는 값을 받아오기만 할 뿐, 받아온 props 를 직접 수정할 수는 없음부모 Component에서 props를 전달해주면 자식 Component는 [ this.[propsName] ]으로 접근할 수 있다

// 부모 Comp <ChildComponent name="이름" /> // 자식 Comp let varialble1 = this.props.name; console.log(variable1); // 이름

defaultProps

props가 빈 값이어야 하는 경우나, 실수로 빼먹어서 오류가 날 수도 있기 때문에 props의 기본값을 설정함 Component class 선언 내부에서

static defaultProps = { name : '기본 이름' }

또는, Component class 선언 외부에서

CompName.defaultProps = { // CompName에 현재 Component 명 name : '기본 이름' }

propType 은 v16부터 deprecated (v15.5까지만 허용됨)

State

Component 내부에서 선언하며 내부에서 값을 변경할 수 있으며, 동적인 데이터를 다룬다
